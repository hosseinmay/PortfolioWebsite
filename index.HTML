My main project at Zynga was the development of a multi-variate experimentation platform to replace the existing A/B testing framework. ARDUINO VUMETER
Hardware Project
A project to combine my love for music and hardware, while introducing me to low-level software design. Can read any song from any audio device - provided that a phone connector is used to connect the music player with the Arduino. Makes use of hardware interrupts to recalibrate volume meter to a base level. Developed with a memory limitation of 2KB, in the C programming language.

Links:  GitHub, Video (YouTube Link)
Technologies Used:  Arduino UNO, Arduino IDE and C
